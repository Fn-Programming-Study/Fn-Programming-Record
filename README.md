# 함수형 프로그래밍

## 개요
백엔드와 프론트엔드가 명확하게 분리되기 시작하면서 프론트엔드에서는 객체지향형 프로그래밍 보다는 선언형에 기반한 함수형 프로그래밍 패러다임이 떠오르고 있습니다. 이를 숙지하기 위해 함수형 프로그래밍과 연관된 도서를 독서하고 구성원들과 네트워킹하며 지식을 습득하고자 시작하게 되었습니다.
